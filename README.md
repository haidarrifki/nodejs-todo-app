# Node.js To-do App

This project is simple CRUD To-Do List application built in Node.js and using MongoDB.

## Pre-requisite
1. Install Node.js >= v12.x.
2. Install MongoDB >= v3.6.

## Next steps
- clone this repository
- create `.env` file
- ```npm install -g pm2```
- ```yarn install```
- ```yarn dev```
- open browser `http://localhost:1337`
